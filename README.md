# stage-overlay

Stateless PractiScore stage overlay builder for DaVinci Resolve.

## What it does

- Opens an authenticated PractiScore session with Playwright
- Lets you find recent matches or paste a results URL
- Resolves a preferred shooter for the current session
- Generates one PNG overlay per stage

## Scripts

- `npm run dev`
- `npm run build`
- `npm test`
